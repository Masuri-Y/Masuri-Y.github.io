MySQL安全加固

发表于2019-04-17|更新于2020-11-21|MySQL
|阅读量:

新安装的MySQL没有经过安全加固时，管理员的账户是空口令，匿名的账号也能登录数据库，此时是非常不安全的

匿名用户登录MySQL

当前MySQL数据库为新安装的数据库，所以使用匿名用户可以进行登录

不过好在匿名用户的权限不大无法查看到数据库中的内容

安全起见，新安装的数据库需要进行安全加固

数据库的安全加固

MySQL安装完毕后需要执行mysql_secure_installaion命令来进行安全的加固
