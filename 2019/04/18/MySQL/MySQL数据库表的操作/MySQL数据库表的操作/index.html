数据类型

数据类型大致理解为此数据是数字还是字符串。

数据类型类型不同数据的样子形式可以不同，数据在磁盘上占用的空间大小也不同。

创建表

创建表需要定义有哪些字段，有了字段后就可以在其内部添加一条一条的记录了。

表的结构包括几个关键的属性：

  1. 表里面有哪些字段，字段名是什么
  2. 字段内数据的类型
  3. 修饰符，用来标识字段的特殊属性，比如主键、唯一键、是否允许为空等等。
